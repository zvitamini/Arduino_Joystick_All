# Arduino_Joystick_All
Burada Joystick kullanarak yaptığımız bütün çalışmaların kodları yer alıyor ♥
Dosyalar için tek tek devre şeması çizmedim, oldukça basit bir şekilde kodları okuyarak devreyi kurabilirsiniz. Pinleri kendi istediğiniz şekilde ayarlayabilirsiniz.
Elimden geldiğince joystick kodlarını bu repoya ekleyeceğim.
!!! Lütfen bunu evde deneyin !!!
